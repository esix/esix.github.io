lcc-wasm — компилятор C, который сам является WebAssembly

Сам компилятор LCC (фронтенд + наш wasm-бэкенд) скомпилирован в rcc.wasm. Он компилирует код на C ниже сразу в бинарный .wasm, и браузер тут же его выполняет. Без сервера, без нативного тулчейна, без отдельного ассемблера — только сам компилятор ~280 КБ.
